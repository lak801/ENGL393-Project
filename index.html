Introduction

Welcome! At the end of this guide, you will be able to make a website of your very own with html and css.
Although this website will be basic, you can use it as an introduction to the world of web development!

Some quick information:

  • HTML stands for Hyper Text Markup Language. This is a fancy name for a computer language that allows you to display text in a web browser.
  • CSS stands for Cascading Style Sheets. It goes hand in hand with HTML because it gives it the "style" that allows for colors and customization.
  • An HTML tag looks like <p>. There are many html tags that allow web developers to format text and include anything from photos to tables. The format for the tags is <Your Tag>.

Setting up the Environment

To start, let's open the text editor on your computer.

For Mac: click your Spotlight application and type in TextEdit and open it.
For Windows: click your Start button and type in Notepad and open it.

You are going to want to save this as an html document:

  • Click file
  • Click save as
  • Click "save as type"
  • Rename the file to "my website.html"
  • And save it to your desktop

Now you should have your very own html file on your desktop ready for you to open, but it will be blank.

Your next step is to right click that html file and click "open with" and then select your text editor so we can get started.



Making the Header

This section will involve a lot of setup for the actual website.

The meaning behind each of these steps is to tell your web browser what to do. Explanations will be provided for each tag as you go along.

This is a reference image for formatting and an example of what your text should look like


  • Type <!DOCTYPE html> to tell your browser you are making an HTML file.
  • Type <html></html> to say that you are going to be using html to write the webpage. Make sure you type in between these html tags
  • Type <head> to create a space that holds information regarding the website
  • Type <title></title> and type in between the two tags. This is an invisible title that tells the name of the webpage.
  • To set some hidden information that will make the website more viewable, type these two lines <meta charset="UTF-8"> and <meta name="viewport" content="width=device-width, initial-scale=1">
  • To set up CSS styling in the HTML you want to type <style></style> which we will use later
  • Finally close your header tag with </header>

In development, keeping formatting will be helpful for readability!



Making the Body

Now, we will be making the stuff that people can actually see

Here is another reference just in case you get lost!



  • Type <body></body> after the closing </head> tag but before the closing </html> tag.
  • Make an <h1> tag between the opening <body> and closing </body>, type what you want to be displayed, and then close the tag with </h1>.
    • h tags are known as header tags because they come in 6 different sizes: h1 through h6, and are mainly used for headings and subtitles.
  • Next, add a <p></p> to create a "paragraph" tag that will allow you to write plain text between it.
